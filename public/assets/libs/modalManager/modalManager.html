Documentation pour les développeurs : ModalManager

Le ModalManager est un gestionnaire de modals pour les pages web, conçu pour être hautement configurable et facile à utiliser. Vous trouverez ci-dessous les détails pour l'intégration, la configuration, et l'utilisation de ModalManager dans vos projets.

Initialisation

Pour initialiser ModalManager, créez une nouvelle instance de la classe avec les sélecteurs de modal et d'overlay :

const modalManager = new ModalManager('#modal', '#overlay');

Paramètres de configuration

Paramètre Type Description
modalSelector string Sélecteur CSS pour le modal à manipuler.
overlaySelector string Sélecteur CSS pour l'overlay du modal.

Fonctionnalités et Méthodes

Méthode Description
show(options) Affiche le modal avec une animation et charge le contenu via AJAX. Les options incluent url, formDataObj, method, animation, duration, onSuccess, et onError.
hide(options) Cache le modal avec une animation spécifiée. Les options incluent animation, duration, et onComplete.
submitForm(options) Soumet un formulaire via AJAX et cache le modal après un succès. Les options incluent url, formDataObj, method, animation, duration, onSuccess, et onError.
cleanup() Nettoie le contenu du modal et cache l'overlay.

1. Affichage du Modal

La méthode show() permet d'afficher un modal en utilisant une URL pour charger du contenu via AJAX. Vous pouvez personnaliser l'animation et la durée de l'affichage en fonction de vos besoins.

2. Masquage du Modal

Pour cacher le modal, utilisez la méthode hide(). Cette méthode supporte différents types d'animations (fade out, slide up, etc.) pour rendre la transition fluide.

3. Soumission d'un Formulaire

La méthode submitForm() permet de soumettre un formulaire directement depuis le modal via une requête AJAX. Si la soumission est réussie, le modal est caché automatiquement.

Exemple Complet d'Utilisation

Voici un exemple complet d'utilisation de ModalManager sur une page HTML :

<div id="overlay" class="hidden"></div>
<div id="modal" class="hidden"></div>
<button onclick="openModal()">Ouvrir le Modal</button>
<script>
    const modalManager = new ModalManager('#modal', '#overlay');
    
    function openModal() {
        modalManager.show({
            url: 'modal-content.html',
            method: 'GET',
            animation: 'fadeIn',
            duration: 800,
            onSuccess: (response) => {
                console.log('Contenu chargé avec succès');
            },
            initialize: () => {
                console.log('Modal initialisé avec les nouvelles fonctionnalités.');
            }
        });
    }
</script>

Styles CSS pour les Modals

Pour que les modals s'affichent correctement, ajoutez les styles CSS suivants :

.hidden {
    display: none;
}

#overlay {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    background-color: rgba(0, 0, 0, 0.5);
    z-index: 999;
}

#modal {
    position: fixed;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    width: 50%;
    background-color: white;
    border-radius: 8px;
    padding: 20px;
    box-shadow: 0px 4px 8px rgba(0, 0, 0, 0.2);
    z-index: 1000;
}

Ces styles définissent l'apparence de l'overlay et du modal. Vous pouvez les personnaliser pour qu'ils s'intègrent parfaitement au thème de votre site.

Remarques

  • Personnalisation des animations : Le paramètre animation permet de choisir parmi plusieurs types d'animations, comme fadeIn, slideDown, etc.
  • Gestion des erreurs : En cas d'erreur lors du chargement du contenu, la méthode onError est appelée et peut être personnalisée.

Avec cette documentation, vous disposez de toutes les informations nécessaires pour utiliser et personnaliser ModalManager dans vos projets web.

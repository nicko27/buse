TableFlow - Documentation

Installation

Pour utiliser TableFlow, il suffit d'inclure le fichier JavaScript principal :

<link rel="stylesheet" href="path/to/tableFlow.css">
<script src="path/to/tableFlow.js"></script>
Les plugins sont chargés automatiquement depuis le répertoire des plugins. Il n'est pas nécessaire de les inclure manuellement.

Initialisation

Initialisez TableFlow en créant une nouvelle instance :

const table = new TableFlow({
    tableId: 'myTable',
    plugins: {
        names: ['Sort', 'Filter', 'Color'],  // Liste des plugins à activer
        color: {  // Configuration spécifique du plugin Color
            customClass: 'glass'
        }
    }
});

Attributs disponibles

Attribut Description Exemple
th-sort Active le tri sur une colonne <th th-sort>
th-filter Active le filtrage sur une colonne <th th-filter>
th-color Active le sélecteur de couleur sur une colonne <th th-color>
th-edit Active l'édition sur une colonne <th th-edit>
th-validate Ajoute des règles de validation <th th-validate='{"required": true}'>
th-actions Ajoute des actions (supprimer, sauvegarder) <th th-actions="delete,save">

Plugins disponibles

Color

color.js

Plugin de sélection de couleur avec prévisualisation et support des classes personnalisées.

Attributs

th-color Active le sélecteur de couleur sur une colonne

Configuration

const table = new TableFlow({
    plugins: {
        names: ['Color'],
        color: {
            colorAttribute: 'th-color',
            customClass: 'glass'  // Classe personnalisée pour le style
        }
    }
});
La classe personnalisée (customClass) est appliquée à tous les composants du sélecteur de couleur : preview, picker, canvas et presets.

Sort

sort.js

Plugin de tri des colonnes avec support multi-colonnes.

Attributs

th-sort Active le tri sur une colonne
th-sort-type Type de tri (text, number, date)

Configuration

const table = new TableFlow({
    plugins: {
        names: ['Sort'],
        sort: {
            showIcons: true,
            icons: {
                asc: '↑',
                desc: '↓',
                none: '↕'
            }
        }
    }
});

Filter

filter.js

Plugin de filtrage avancé avec support de multiples types de filtres.

Attributs

th-filter Active le filtrage sur une colonne
th-filter-type Type de filtre (text, select, date)

Edit

edit.js

Plugin d'édition en ligne avec validation et sauvegarde automatique.

Attributs

th-edit Active l'édition sur une colonne
th-validate Règles de validation pour l'édition

Actions

actions.js

Plugin d'actions sur les lignes (supprimer, sauvegarder).

Attributs

th-actions Liste des actions disponibles

Configuration

const table = new TableFlow({
    plugins: {
        names: ['Actions'],
        actions: {
            useIcons: true,
            icons: {
                delete: '<i class="fa fa-trash"></i>',
                save: '<i class="fa fa-save"></i>'
            }
        }
    }
});

Personnalisation

TableFlow supporte la personnalisation via des classes CSS personnalisées :

const table = new TableFlow({
    plugins: {
        names: ['Color', 'Actions'],
        color: {
            customClass: 'glass'  // Effet de verre pour le sélecteur de couleur
        },
        actions: {
            buttonClass: 'custom-button',  // Style personnalisé pour les boutons d'action
            containerClass: 'custom-container'
        }
    }
});
Les classes personnalisées peuvent être utilisées pour appliquer des effets visuels cohérents à travers l'interface.

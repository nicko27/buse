Introduction

TableFlow est une bibliothèque JavaScript qui permet de créer des tableaux interactifs avec des fonctionnalités avancées comme l'édition en ligne, le tri, la pagination, et plus encore.

Installation

Pour utiliser TableFlow, importez simplement le module dans votre projet :

import TableFlow from './path/to/tableFlow.js';

Configuration de base

Voici un exemple de configuration basique :

const table = new TableFlow({
    tableId: 'monTableau',
    debug: false,
    cellWrapper: true,
    plugins: {
        // Configuration des plugins
    }
});

Options principales

Option Type Description
tableId string ID de l'élément table HTML
debug boolean Active/désactive le mode debug
cellWrapper boolean Ajoute un wrapper autour des cellules

Plugins disponibles

Plugin d'édition (edit.js)

Permet l'édition directe des cellules du tableau.

edit: {
    path: '../plugins/edit.js'
}

Plugin de tri (sort.js)

Ajoute la fonctionnalité de tri sur les colonnes.

sort: {
    path: '../plugins/sort.js',
    config: {
        sortableAttribute: 'th-sort'
    }
}

Plugin de couleur (color.js)

Permet la gestion des couleurs dans les cellules.

color: {
    path: '../plugins/color.js',
    config: {
        colorAttribute: 'th-color',
        customClass: 'glass'
    }
}

Plugin d'actions (actions.js)

Ajoute des boutons d'action pour chaque ligne.

actions: {
    path: '../plugins/actions.js',
    config: {
        actionAttribute: 'th-actions',
        cellClass: 'td-actions',
        useIcons: true,
        actions: {
            delete: (context) => handleAction(context, 'delete'),
            save: (context) => handleAction(context, 'save')
        }
    }
}

Plugin de filtrage et pagination (filterandpaginate.js)

Ajoute des fonctionnalités de filtrage et pagination au tableau.

filterAndPaginate: {
    path: '../plugins/filterandpaginate.js',
    config: {
        itemsPerPage: 10,
        filterAttribute: 'th-filter'
    }
}

Plugin de sélection (select.js)

Permet la sélection de lignes avec des cases à cocher.

select: {
    path: '../plugins/select.js',
    config: {
        selectableAttribute: 'th-select',
        multiSelect: true
    }
}

Exemples d'utilisation

Configuration complète

const ACTIONS_URL = "/main/actions.php";
const PLUGINS_BASE_PATH = '../plugins/';

const handleAction = (context, action, module = "default") => {
    actionRow(ACTIONS_URL, context.data, module, action)
        .then(result => {
            if (result) {
                successNotice('Ligne mise à jour avec succès');
                context.tableHandler.markRowAsSaved(context.row, action);
            }
        })
        .catch(error => console.error('Error:', error));
};

const table = new TableFlow({
    tableId: 'monTableau',
    debug: false,
    cellWrapper: true,
    plugins: {
        edit: {
            path: PLUGINS_BASE_PATH + 'edit.js'
        },
        sort: {
            path: PLUGINS_BASE_PATH + 'sort.js',
            config: {
                sortableAttribute: 'th-sort'
            }
        },
        color: {
            path: PLUGINS_BASE_PATH + 'color.js',
            config: {
                colorAttribute: 'th-color'
            }
        },
        actions: {
            path: PLUGINS_BASE_PATH + 'actions.js',
            config: {
                actionAttribute: 'th-actions',
                useIcons: true,
                actions: {
                    delete: (context) => handleAction(context, 'delete'),
                    save: (context) => handleAction(context, 'save')
                }
            }
        }
    }
});

Structure HTML correspondante

<table id="monTableau">
    <thead>
        <tr>
            <th th-edit th-sort>Nom</th>
            <th th-edit th-color>Couleur</th>
            <th th-actions="delete,save">Actions</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Exemple</td>
            <td>#FF0000</td>
            <td></td>
        </tr>
    </tbody>
</table>

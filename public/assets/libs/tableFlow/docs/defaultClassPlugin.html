Documentation DefaultClassPlugin - nvTblHandler

Le plugin DefaultClassPlugin permet d'appliquer automatiquement des classes CSS par défaut aux éléments du tableau. Il facilite la gestion des styles et assure une apparence cohérente.

Caractéristiques Principales

  • Application automatique de classes CSS
  • Classes personnalisables par élément
  • Support des états (hover, focus, active)
  • Classes conditionnelles
  • Gestion des lignes paires/impaires
  • Support des thèmes
  • Classes responsives

Configuration

Exemple de configuration du plugin :

const table = new NvTblHandler({
    tableId: 'monTableau',
    plugins: {
        DefaultClass: {
            table: 'th-table',
            header: 'th-header',
            headerCell: 'th-header-cell',
            body: 'th-body',
            row: 'th-row',
            cell: 'th-cell',
            footer: 'th-footer',
            footerCell: 'th-footer-cell',
            striped: true,
            stripedClass: 'th-striped',
            hoverClass: 'th-hover',
            activeClass: 'th-active',
            selectedClass: 'th-selected',
            responsiveClasses: {
                sm: 'th-sm',
                md: 'th-md',
                lg: 'th-lg'
            },
            theme: 'default',
            themes: {
                default: {
                    table: 'th-theme-default',
                    header: 'th-theme-header'
                },
                dark: {
                    table: 'th-theme-dark',
                    header: 'th-theme-dark-header'
                }
            },
            conditionalClasses: {
                row: (row, index) => {
                    return {
                        'th-highlight': index === 0,
                        'th-modified': row.hasAttribute('data-modified')
                    };
                },
                cell: (cell, column) => {
                    return {
                        'th-numeric': column.type === 'number',
                        'th-date': column.type === 'date'
                    };
                }
            }
        }
    }
});

Options de Configuration

Option Type Description Défaut
table string Classe pour l'élément table 'th-table'
header string Classe pour l'en-tête 'th-header'
headerCell string Classe pour les cellules d'en-tête 'th-header-cell'
body string Classe pour le corps du tableau 'th-body'
row string Classe pour les lignes 'th-row'
cell string Classe pour les cellules 'th-cell'
striped boolean Activer les lignes alternées true
stripedClass string Classe pour les lignes alternées 'th-striped'
theme string Thème actif 'default'

Méthodes Publiques

Méthode Arguments Description Retour
init(tableHandler) tableHandler: object Initialise le plugin void
setTheme(theme) theme: string Change le thème void
refresh() - Rafraîchit les classes void
destroy() - Nettoie le plugin void

Structure HTML

Exemple de structure HTML avec classes par défaut :

<table id="monTableau" class="th-table th-theme-default">
    <thead class="th-header">
        <tr class="th-row">
            <th class="th-header-cell">ID</th>
            <th class="th-header-cell">Nom</th>
            <th class="th-header-cell">Email</th>
            <th class="th-header-cell">Date</th>
        </tr>
    </thead>
    <tbody class="th-body">
        <tr class="th-row">
            <td class="th-cell th-numeric">1</td>
            <td class="th-cell">Jean Dupont</td>
            <td class="th-cell">jean@example.com</td>
            <td class="th-cell th-date">2023-01-15</td>
        </tr>
        <tr class="th-row th-striped">
            <td class="th-cell th-numeric">2</td>
            <td class="th-cell">Marie Martin</td>
            <td class="th-cell">marie@example.com</td>
            <td class="th-cell th-date">2023-01-16</td>
        </tr>
    </tbody>
</table>

Styles CSS

Styles recommandés pour les classes par défaut :

/* Classes de base */
.th-table {
    width: 100%;
    border-collapse: collapse;
    margin-bottom: 1rem;
}

.th-header {
    background-color: #f8f9fa;
}

.th-header-cell {
    padding: 0.75rem;
    border-bottom: 2px solid #dee2e6;
    font-weight: bold;
    text-align: left;
}

.th-body {
    background-color: #fff;
}

.th-row {
    border-bottom: 1px solid #dee2e6;
}

.th-cell {
    padding: 0.75rem;
}

/* Classes d'état */
.th-striped {
    background-color: rgba(0, 0, 0, 0.05);
}

.th-hover:hover {
    background-color: rgba(0, 0, 0, 0.075);
}

.th-active {
    background-color: rgba(0, 123, 255, 0.1);
}

.th-selected {
    background-color: rgba(0, 123, 255, 0.2);
}

/* Classes de type */
.th-numeric {
    text-align: right;
}

.th-date {
    text-align: center;
}

/* Classes responsives */
@media (max-width: 576px) {
    .th-sm {
        font-size: 0.875rem;
    }
}

@media (max-width: 768px) {
    .th-md {
        font-size: 0.9375rem;
    }
}

/* Thèmes */
.th-theme-dark {
    background-color: #343a40;
    color: #fff;
}

.th-theme-dark .th-header {
    background-color: #212529;
}

.th-theme-dark .th-striped {
    background-color: rgba(255, 255, 255, 0.05);
}

Thèmes Personnalisés

Exemple de configuration de thèmes personnalisés :

themes: {
    // Thème clair
    light: {
        table: 'th-theme-light',
        header: 'th-theme-light-header',
        row: 'th-theme-light-row',
        striped: 'th-theme-light-striped'
    },

    // Thème sombre
    dark: {
        table: 'th-theme-dark',
        header: 'th-theme-dark-header',
        row: 'th-theme-dark-row',
        striped: 'th-theme-dark-striped'
    },

    // Thème compact
    compact: {
        table: 'th-theme-compact',
        cell: 'th-theme-compact-cell',
        headerCell: 'th-theme-compact-header'
    }
}

Notes Importantes

  • Les classes sont appliquées automatiquement à l'initialisation
  • Les thèmes peuvent être changés dynamiquement
  • Les classes conditionnelles sont réévaluées lors des modifications
  • Les classes responsives s'adaptent à la taille de l'écran
  • Les classes personnalisées peuvent surcharger les classes par défaut

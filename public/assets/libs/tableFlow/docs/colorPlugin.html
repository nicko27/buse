Plugin Color

Le plugin Color ajoute des fonctionnalités de sélection et de gestion de couleurs à votre tableau. Il transforme les cellules en sélecteurs de couleurs interactifs avec prévisualisation et historique des modifications.

Installation

Incluez le fichier du plugin après TableFlow :

<script src="plugins/color.js"></script>

Configuration

Configuration du plugin dans TableFlow :

const table = new TableFlow({
    tableId: 'monTableau',
    plugins: {
        names: ['Color'],
        color: {
            colorAttribute: 'th-color',
            customClass: 'glass'  // Applique un effet visuel personnalisé
        }
    }
});

Options disponibles

Option Type Défaut Description
colorAttribute string 'th-color' Attribut HTML pour identifier les colonnes avec sélecteur de couleur
customClass string '' Classe CSS appliquée à tous les composants du sélecteur de couleur (preview, picker, canvas, presets)

Utilisation

Ajoutez l'attribut th-color aux en-têtes de colonnes concernées :

<table id="monTableau">
    <thead>
        <tr>
            <th>Nom</th>
            <th th-color>Couleur</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Element 1</td>
            <td>#FF0000</td>
        </tr>
    </tbody>
</table>

Composants

Composant Classe CSS Description
Aperçu de couleur .color-preview Rectangle affichant la couleur sélectionnée
Input de couleur .color-input Champ texte affichant la valeur hexadécimale
Sélecteur de couleur .nv-color-picker Popup contenant le sélecteur de couleur
Canvas de sélection .nv-color-canvas Zone de sélection de couleur interactive
Couleurs prédéfinies .nv-color-presets Grille des couleurs prédéfinies
Bouton de préréglage .nv-color-preset Bouton de couleur prédéfinie individuel
Note : La classe customClass spécifiée dans la configuration sera appliquée à tous ces composants, permettant une personnalisation cohérente de l'interface.

Événements

Événement Données Description
colorChange { oldColor, newColor, cell } Déclenché lorsqu'une couleur est modifiée

Compatibilité

Le plugin Color est compatible avec les plugins suivants :

  • Edit : Permet l'édition directe des valeurs de couleur
  • Sort : Permet le tri des colonnes par couleur
  • Filter : Permet le filtrage des colonnes par couleur
  • Validate : Supporte la validation des valeurs de couleur

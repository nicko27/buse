Documentation ColorFlow

Introduction

ColorFlow est une bibliothèque JavaScript simple et légère pour ajouter un sélecteur de couleur interactif à vos champs de formulaire HTML. Elle offre un aperçu de la couleur sélectionnée, un sélecteur visuel (canvas) et une liste de couleurs prédéfinies.

Caractéristiques principales :

  • Facile à intégrer avec un simple attribut HTML.
  • Aperçu de la couleur à côté de l'input.
  • Sélecteur de couleur basé sur un canvas (Teinte/Saturation/Luminosité).
  • Palette de couleurs prédéfinies personnalisable.
  • Détection automatique des nouveaux champs ajoutés dynamiquement au DOM.
  • Validation basique du format de couleur (hexadécimal).

Installation

Pour utiliser ColorFlow, incluez son fichier CSS et son fichier JavaScript dans votre page HTML.

<!-- Feuille de style de ColorFlow -->
<link rel="stylesheet" href="path/to/colorFlow.css">

<!-- Script de ColorFlow -->
<script src="path/to/colorFlow.js"></script>
Assurez-vous que les chemins d'accès aux fichiers colorFlow.css et colorFlow.js sont corrects.

Initialisation JavaScript

Vous pouvez initialiser ColorFlow en créant une nouvelle instance de la classe ColorFlow. Cela vous permet de configurer des options globales, comme les couleurs prédéfinies.

// Initialisation après le chargement du DOM
document.addEventListener('DOMContentLoaded', () => {
    const colorFlow = new ColorFlow({
        // Options de configuration globales (voir ci-dessous)
        presetColors: [
            { color: '#FF0000', label: 'Rouge' },
            { color: '#00FF00', label: 'Vert' },
            { color: '#0000FF', label: 'Bleu' },
            { color: '#FFFF00', label: 'Jaune' },
            { color: '#FF00FF', label: 'Magenta' },
            { color: '#00FFFF', label: 'Cyan' },
            { color: '#000000', label: 'Noir' },
            { color: '#FFFFFF', label: 'Blanc' }
        ],
        customClass: 'mon-theme-couleur', // Classe CSS additionnelle
        debug: false // Activer les logs de debug
    });

    // Optionnel: Écouter les changements de valeur
    document.body.addEventListener('change', (event) => {
        if (event.target.hasAttribute('cf-input')) {
            console.log('Nouvelle couleur sélectionnée:', event.target.value);
            // Vous pouvez ajouter ici du code pour réagir au changement
        }
    });
});

Même si vous n'avez pas besoin de personnaliser les options, la création d'une instance new ColorFlow() est recommandée pour lancer la détection initiale des champs.

Options de Configuration (Constructeur)

Option Type Défaut Description
presetColors Array<Object> Liste de 8 couleurs de base Tableau d'objets pour définir les couleurs prédéfinies. Chaque objet doit avoir les clés color (valeur hex) et label (texte court affiché sur la couleur).
customClass string '' Classe CSS personnalisée à ajouter aux éléments du sélecteur (preview, picker, canvas, presets) pour un style spécifique.
debug boolean false Active les messages de débogage dans la console.

Utilisation HTML

Pour transformer un champ texte en sélecteur de couleur ColorFlow, ajoutez simplement l'attribut cf-input à votre élément <input type="text">.

<input type="text" cf-input />

ColorFlow ajoutera automatiquement un aperçu cliquable avant l'input et gérera l'affichage du sélecteur popup.

Attributs HTML (Configuration par Élément)

Vous pouvez personnaliser le comportement de chaque sélecteur via des attributs data- sur l'élément input :

Attribut Description Valeurs Possibles Exemple
cf-input Requis. Active ColorFlow pour cet input. N/A (présence de l'attribut) <input cf-input>
cf-default Définit la couleur par défaut si l'input est vide ou invalide. Une couleur valide (ex: #FF0000, red) <input cf-input cf-default="#FFA500">
cf-format (Fonctionnalité limitée) Format de couleur attendu. hex (seul format réellement géré en sortie) <input cf-input cf-format="hex">
cf-alpha (Non implémenté) Active la sélection de la transparence (canal alpha). true / false <input cf-input cf-alpha>
cf-palette (Pas d'effet visible) Indique l'utilisation d'une palette. true / false <input cf-input cf-palette>
Note sur les attributs : Actuellement, les attributs cf-format, cf-alpha, et cf-palette sont lus par le script mais n'ont pas d'impact fonctionnel majeur sur le sélecteur ou la valeur retournée (qui est toujours en hexadécimal sans alpha).
Défaut Orange: Valeur initiale Rouge: 
<!-- Couleur par défaut orange -->
<input type="text" cf-input cf-default="#FFA500" placeholder="#FFA500"/>

<!-- Input avec une valeur initiale rouge -->
<input type="text" cf-input value="#FF0000" />

Fonctionnement

  • Détection Automatique : ColorFlow utilise un `MutationObserver` pour détecter automatiquement les nouveaux éléments <input cf-input> ajoutés au DOM après l'initialisation, les rendant fonctionnels sans appel manuel.
  • Aperçu Couleur (`.color-preview`) : Un élément `div` avec la classe `.color-preview` est inséré avant l'input. Son arrière-plan reflète la couleur actuelle de l'input. Cliquer dessus ouvre le sélecteur.
  • Sélecteur Popup (`.nv-color-picker`) :
    • S'affiche en cliquant sur l'aperçu ou l'input.
    • Contient un canvas (`.nv-color-canvas`) pour une sélection visuelle de la teinte, saturation et luminosité.
    • Affiche une grille (`.nv-color-presets`) de couleurs prédéfinies (configurables via l'option `presetColors`).
    • Se ferme automatiquement en cliquant en dehors ou en sélectionnant une couleur.
  • Validation : Lorsque la valeur de l'input change (manuellement ou via le sélecteur), ColorFlow vérifie si la couleur est valide. Si elle est invalide, la valeur revient à la couleur précédente valide. La sortie est toujours en format hexadécimal (ex: `#FF0000`).

Personnalisation CSS

Vous pouvez personnaliser l'apparence de ColorFlow en ciblant les classes CSS suivantes :

  • .color-preview: L'aperçu carré de la couleur.
  • .nv-color-picker: Le conteneur principal du popup.
  • .nv-color-canvas: Le canvas de sélection de couleur.
  • .nv-color-presets: Le conteneur de la grille des couleurs prédéfinies.
  • .nv-color-preset: Chaque bouton de couleur prédéfinie.

De plus, la classe fournie dans l'option de configuration customClass sera ajoutée à ces éléments, vous permettant de créer des thèmes spécifiques.

/* Exemple : Thème personnalisé 'mon-theme-couleur' */
.nv-color-picker.mon-theme-couleur {
    border: 2px solid navy;
}

.color-preview.mon-theme-couleur {
    border-radius: 50%; /* Rendre l'aperçu rond */
}

.nv-color-preset.mon-theme-couleur {
    border: 1px dashed grey;
    border-radius: 0; /* Rendre les presets carrés */
}

API JavaScript (Méthodes sur l'instance)

L'instance `ColorFlow` expose quelques méthodes utiles :

Méthode Arguments Description Retour
init() - Initialise ColorFlow (appelé automatiquement par le constructeur). Détecte les inputs existants et configure les observateurs. void
refresh() - Recherche et initialise les nouveaux inputs [cf-input] qui n'ont pas encore été traités (utile si vous ajoutez des inputs après le chargement initial sans utiliser l'observateur DOM). void
isValidColor(color) color (string) Vérifie si une chaîne de caractères représente une couleur CSS valide. boolean
toHexColor(color) color (string) Tente de convertir une couleur (nom, rgb, etc.) en format hexadécimal majuscule (#RRGGBB). Retourne #000000 en cas d'échec. string
setCurrentColor(color) color (string) Met en surbrillance (classe 'selected') le bouton de couleur prédéfinie correspondant à la couleur donnée. Ne change pas la couleur active du sélecteur canvas. void
destroy() - Supprime les écouteurs d'événements globaux et le picker du DOM. Ne retire pas les aperçus des inputs. void

Exemple Complet

Cet exemple montre comment initialiser ColorFlow avec des couleurs prédéfinies personnalisées et écouter les changements.

Texte d'aperçu
<!-- HTML -->
<label for="couleur-fond">Couleur de fond:</label>
<input type="text" id="couleur-fond" cf-input cf-default="#EEEEEE" value="#EEEEEE" />

<label for="couleur-texte">Couleur du texte:</label>
<input type="text" id="couleur-texte" cf-input cf-default="#333333" value="#333333"/>

<div id="apercu-style" style="padding: 10px; border: 1px solid black;">
    Texte d'aperçu
</div>

<!-- JavaScript -->
<script>
document.addEventListener('DOMContentLoaded', () => {
    const customPresets = [
        { color: '#D9E3F0', label: 'Bleu pâle' }, { color: '#F47373', label: 'Rouge doux' },
        { color: '#697689', label: 'Gris bleu' }, { color: '#37D67A', label: 'Vert vif' },
        { color: '#2CCCE4', label: 'Cyan vif' }, { color: '#555555', label: 'Gris foncé' },
        { color: '#dce775', label: 'Vert lime' }, { color: '#ff8a65', label: 'Orange doux' },
        { color: '#ba68c8', label: 'Violet doux' }
    ];

    const colorFlow = new ColorFlow({
        presetColors: customPresets
    });

    const bgInput = document.getElementById('couleur-fond');
    const textInput = document.getElementById('couleur-texte');
    const previewDiv = document.getElementById('apercu-style');

    // Fonction pour mettre à jour le style de l'aperçu
    function updatePreviewStyle() {
        previewDiv.style.backgroundColor = bgInput.value;
        previewDiv.style.color = textInput.value;
    }

    // Écouter les changements sur les inputs de couleur
    bgInput.addEventListener('change', updatePreviewStyle);
    textInput.addEventListener('change', updatePreviewStyle);

    // Appliquer le style initial
    updatePreviewStyle();
});
</script>

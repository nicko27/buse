Documentation ColorFlow

Introduction

ColorFlow est un gestionnaire de sélection de couleur simple et efficace pour les applications web. Il offre une interface intuitive pour la sélection des couleurs avec un support pour les couleurs prédéfinies et une palette de couleurs personnalisable.

Installation

Pour utiliser ColorFlow, incluez les fichiers CSS et JS dans votre page HTML :

<link rel="stylesheet" href="/assets/libs/colorFlow/src/colorFlow.css" />
<script src="/assets/libs/colorFlow/src/colorFlow.js"></script>

Utilisation de base

Pour créer un sélecteur de couleur, ajoutez l'attribut cf-input à votre input :

<input type="text" cf-input />

Configuration

ColorFlow peut être configuré avec les options suivantes :

const colorFlow = new ColorFlow({
    presetColors: [
        { color: '#FF0000', label: 'RED' },
        { color: '#00FF00', label: 'GRN' },
        // ... autres couleurs
    ],
    customClass: '', // classe CSS personnalisée
    debug: false     // mode debug
});

Attributs disponibles

  • cf-input : Active le sélecteur de couleur sur l'input
  • cf-format : Format de la couleur (hex par défaut)
  • cf-alpha : Active le support de la transparence
  • cf-palette : Active la palette de couleurs
  • cf-default : Couleur par défaut (#000000 si non spécifié)

Styles CSS

ColorFlow inclut plusieurs classes CSS personnalisables :

  • .color-flow : Conteneur principal
  • .color-preview : Aperçu de la couleur
  • .nv-color-picker : Conteneur du sélecteur de couleur
  • .nv-color-canvas : Canvas pour la sélection de couleur
  • .nv-color-presets : Section des couleurs prédéfinies
  • .nv-color-preset : Bouton de couleur prédéfinie

Note sur l'accessibilité

ColorFlow supporte la navigation au clavier et inclut des étiquettes pour les lecteurs d'écran. Les couleurs peuvent être entrées manuellement dans l'input en format hexadécimal.

Méthodes disponibles

  • init() : Initialise le sélecteur de couleur
  • destroy() : Nettoie et supprime le sélecteur de couleur
  • setCurrentColor(color) : Définit la couleur actuelle
  • isValidColor(color) : Vérifie si une couleur est valide

Exemple complet

<!-- HTML -->
<input type="text" 
       cf-input 
       cf-format="hex" 
       cf-default="#FF0000" 
       value="#FF0000" />

<!-- JavaScript -->
<script>
const colorFlow = new ColorFlow({
    presetColors: [
        { color: '#FF0000', label: 'RED' },
        { color: '#00FF00', label: 'GRN' },
        { color: '#0000FF', label: 'BLU' }
    ],
    customClass: 'my-theme'
});

// Écouter les changements de couleur
document.querySelector('[cf-input]').addEventListener('change', (e) => {
    console.log('Nouvelle couleur:', e.target.value);
});
</script>

NotifyFlow Demo

Configuration

Notifications basiques

notify.success('Opération réussie !');
notify.error('Une erreur est survenue');
notify.warning('Attention !');
notify.info('Information importante');

Notifications avec titre

notify.success('Message', {
    title: 'Titre',
    duration: 5000
});

Notifications avec actions

notify.create({
    title: 'Actions',
    message: 'Choisissez une action',
    actions: [
        {
            id: 'action1',
            text: 'Action 1',
            callback: () => {}
        }
    ]
});

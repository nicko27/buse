NotifyFlow - Documentation

Installation

Pour utiliser NotifyFlow, incluez les fichiers CSS et JavaScript dans votre page HTML :

<link rel="stylesheet" href="path/to/notifyFlow.css">
<script src="path/to/notifyFlow.js"></script>

Initialisation

Initialisez NotifyFlow en créant une nouvelle instance :

const notify = new NotifyFlow();

Options disponibles

Option Description Valeurs possibles Défaut
type Type de notification success, error, warning, info info
position Position de la notification top-right, top-left, bottom-right, bottom-left top-right
duration Durée d'affichage (ms) nombre ou 'infinite' 3000
dismissible Peut être fermée true, false true
animation Type d'animation fade, slide, bounce fade

Exemples d'utilisation

1. Notifications simples

notify.show('Opération réussie !', 'success');
notify.show('Une erreur est survenue.', 'error');
notify.show('Attention !', 'warning');
notify.show('Information importante.', 'info');

2. Positions différentes

notify.show('Message', 'info', { position: 'top-right' });
notify.show('Message', 'info', { position: 'top-left' });
notify.show('Message', 'info', { position: 'bottom-right' });
notify.show('Message', 'info', { position: 'bottom-left' });

3. Durées personnalisées

notify.show('Message', 'info', { duration: 1000 });
notify.show('Message', 'info', { duration: 5000 });
notify.show('Message', 'info', { duration: 'infinite' });

4. Animations différentes

notify.show('Message', 'info', { animation: 'fade' });
notify.show('Message', 'info', { animation: 'slide' });
notify.show('Message', 'info', { animation: 'bounce' });

5. Notification avec HTML

notify.show('<b>Texte en gras</b> et <i>italique</i>', 'info', { html: true });

API JavaScript

// Initialisation avec options globales
const notify = new NotifyFlow({
    position: 'top-right',
    duration: 3000,
    animation: 'fade'
});

// Méthodes disponibles
notify.show(message, type, options);  // Affiche une notification
notify.success(message, options);     // Raccourci pour type success
notify.error(message, options);       // Raccourci pour type error
notify.warning(message, options);     // Raccourci pour type warning
notify.info(message, options);        // Raccourci pour type info
notify.clear();                       // Efface toutes les notifications

Notes importantes

1. Les notifications sont automatiquement empilées.

2. Les notifications peuvent être fermées en cliquant dessus (si dismissible: true).

3. Le HTML est désactivé par défaut pour des raisons de sécurité.

4. Les animations peuvent être désactivées en définissant animation: false.

Documentation NotifyFlow

NotifyFlow est une bibliothèque moderne de notifications pour les applications web, offrant une expérience utilisateur riche et personnalisable.

Installation

Incluez les fichiers CSS et JavaScript dans votre page HTML :

<link rel="stylesheet" href="notifyFlow.css">
<script src="notifyFlow.js"></script>

Configuration

Initialisez NotifyFlow avec vos options personnalisées :

const notifyFlow = new NotifyFlow({
    position: 'top-right',      // Position des notifications
    theme: 'light',            // Thème (light, dark, auto)
    maxNotifications: 5,       // Nombre maximum de notifications
    defaultDuration: 5000,     // Durée par défaut (ms)
    containerClass: 'notify-flow', // Classe du conteneur
    animations: {
        in: 'slide-in',        // Animation d'entrée
        out: 'fade-out'        // Animation de sortie
    }
});

Options de configuration

Option Type Défaut Description
position string 'top-right' Position des notifications (top-right, top-left, bottom-right, bottom-left, top-center, bottom-center)
theme string 'light' Thème des notifications (light, dark, auto)
maxNotifications number 5 Nombre maximum de notifications affichées simultanément
defaultDuration number 5000 Durée d'affichage par défaut en millisecondes

Utilisation

Créez une notification avec différentes options :

notifyFlow.notify({
    title: 'Titre',           // Titre de la notification
    message: 'Message',       // Contenu de la notification
    type: 'info',            // Type (info, success, warning, error)
    icon: '<svg>...</svg>',  // Icône (HTML/SVG)
    duration: 5000,          // Durée d'affichage
    actions: [],             // Actions (boutons)
    progress: true,          // Barre de progression
    dismissible: true,       // Bouton de fermeture
    autoAction: null,        // ID de l'action automatique
    autoActionDelay: 5000,   // Délai avant action auto
    customClass: ''          // Classe CSS personnalisée
});

Exemples d'utilisation

// Notification de succès simple
notifyFlow.notify({
    title: 'Succès',
    message: 'Opération réussie !',
    type: 'success'
});

// Notification avec actions
notifyFlow.notify({
    title: 'Confirmation',
    message: 'Voulez-vous continuer ?',
    type: 'warning',
    duration: 0, // La notification ne disparaîtra pas automatiquement
    actions: [
        {
            id: 'confirm',
            text: 'Confirmer',
            callback: () => console.log('Confirmé'),
            class: 'primary'
        },
        {
            id: 'cancel',
            text: 'Annuler',
            callback: () => console.log('Annulé'),
            class: 'secondary'
        }
    ]
});

Thèmes

NotifyFlow supporte trois modes de thème :

  • light : Thème clair
  • dark : Thème sombre
  • auto : S'adapte aux préférences système

Le thème 'auto' utilise la media query prefers-color-scheme pour détecter les préférences de l'utilisateur.

Actions

Les notifications peuvent inclure des boutons d'action interactifs :

const actions = [
    {
        id: 'action1',          // Identifiant unique
        text: 'Cliquez-moi',    // Texte du bouton
        callback: () => {},     // Fonction de callback
        class: 'custom-class',  // Classe CSS personnalisée
        dismiss: true           // Fermer après clic
    }
];

Actions automatiques

Une action peut être exécutée automatiquement après un délai :

notifyFlow.notify({
    message: 'Redirection dans 5 secondes...',
    autoAction: 'redirect',     // ID de l'action
    autoActionDelay: 5000,     // Délai en ms
    actions: [
        {
            id: 'redirect',
            text: 'Rediriger',
            callback: () => window.location.href = '/dashboard'
        }
    ]
});

Animations

NotifyFlow inclut plusieurs animations prédéfinies :

  • Entrée : slide-in, fade-in
  • Sortie : fade-out, slide-out

Personnalisez les animations via CSS :

/* Animation personnalisée */
@keyframes custom-animation {
    from { transform: scale(0); }
    to { transform: scale(1); }
}

.notify-flow-item.custom-in {
    animation: custom-animation 0.3s ease-out;
}

API

Méthodes

Méthode Description Paramètres
notify(options) Crée une nouvelle notification Object: options de la notification
remove(notification) Supprime une notification spécifique Element: élément DOM de la notification
removeAll() Supprime toutes les notifications -

Types de notifications

Type Usage
info Informations générales
success Opérations réussies
warning Avertissements
error Erreurs

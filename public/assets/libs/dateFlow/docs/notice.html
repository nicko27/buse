DateFlow - Documentation

Installation

Pour utiliser DateFlow, incluez les fichiers CSS et JavaScript dans votre page HTML :

<link rel="stylesheet" href="path/to/dateFlow.css">
<script src="path/to/dateFlow.js"></script>

Initialisation

Initialisez DateFlow en créant une nouvelle instance :

const datePicker = new DateFlow();

Attributs disponibles

Attribut Description Exemple
dp-input Active le DateFlow sur l'input <input dp-input>
dp-format Définit le format de date (YYYY-MM-DD par défaut) <input dp-input dp-format="DD/MM/YYYY">
dp-time Active la sélection de l'heure <input dp-input dp-time>
dp-time-only Active uniquement la sélection de l'heure <input dp-input dp-time-only>
dp-time-interval Définit l'intervalle des minutes (15 par défaut) <input dp-input dp-time dp-time-interval="30">
dp-min Date minimum sélectionnable <input dp-input dp-min="2024-01-01">
dp-max Date maximum sélectionnable <input dp-input dp-max="2024-12-31">
dp-date Date par défaut <input dp-input dp-date="2024-01-01">
dp-show-buttons Affiche les boutons "Aujourd'hui" et "Effacer" <input dp-input dp-show-buttons>

Exemples d'utilisation

1. Date simple

<div class="date-flow">
    <input type="text" dp-input>
</div>

2. Date avec format personnalisé

<div class="date-flow">
    <input type="text" dp-input dp-format="DD/MM/YYYY">
</div>

3. Date et Heure

<div class="date-flow">
    <input type="text" dp-input dp-time>
</div>

4. Heure uniquement (intervalle 30 minutes)

<div class="date-flow">
    <input type="text" dp-input dp-time-only dp-time-interval="30">
</div>

5. Date avec plage limitée

<div class="date-flow">
    <input type="text" dp-input dp-min="2024-01-01" dp-max="2024-12-31">
</div>

6. Date avec valeur par défaut et boutons

<div class="date-flow">
    <input type="text" dp-input dp-date="2024-01-01" dp-show-buttons="true">
</div>

Notes importantes

1. L'élément input doit toujours être à l'intérieur d'un conteneur avec la classe date-flow.

2. L'attribut dp-input est obligatoire pour activer le DateFlow sur un input.

3. Les formats de date supportés sont : YYYY-MM-DD, DD/MM/YYYY, MM/DD/YYYY

4. Pour le sélecteur d'heure, les intervalles possibles sont : 5, 10, 15, 30, 60 minutes

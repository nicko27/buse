TooltipFlow - Documentation

Installation

Pour utiliser TooltipFlow, incluez les fichiers CSS et JavaScript dans votre page HTML :

<link rel="stylesheet" href="path/to/tooltipFlow.css">
<script src="path/to/tooltipFlow.js"></script>

Initialisation

Initialisez TooltipFlow en créant une nouvelle instance. Voici toutes les options disponibles :

const tooltipFlow = new TooltipFlow({
    // Apparence
    theme: 'light',        // 'light', 'dark', ou 'auto'
    animation: 'fade',     // 'fade', 'shift', 'scale', ou 'flip'
    maxWidth: 300,        // Largeur maximale en pixels
    className: '',        // Classes CSS additionnelles
    zIndex: 9999,        // z-index des tooltips

    // Timing
    showDelay: 200,       // Délai avant apparition (ms)
    hideDelay: 200,       // Délai avant disparition (ms)

    // Position
    defaultPosition: 'top', // 'top', 'right', 'bottom', ou 'left'
    offset: 8,            // Distance de l'élément en pixels

    // Comportement
    interactive: false,    // Permet l'interaction avec le tooltip
    allowHTML: false,     // Autorise le HTML dans le contenu
    appendTo: document.body, // Élément parent des tooltips
    
    // Débogage
    debug: false          // Active les logs de débogage
});

Exemples d'utilisation

1. Configuration basique

const tooltipFlow = new TooltipFlow({
    theme: 'light',
    animation: 'fade'
});

2. Configuration interactive

const tooltipFlow = new TooltipFlow({
    theme: 'light',
    animation: 'fade',
    showDelay: 200,
    interactive: true,
    maxWidth: 300,
    offset: 8
});

3. Configuration avec débogage

const tooltipFlow = new TooltipFlow({
    theme: 'light',
    animation: 'fade',
    showDelay: 200,
    debug: true  // Active les logs dans la console
});
Note : L'initialisation doit être faite après le chargement du DOM, soit :
  • Dans un événement DOMContentLoaded
  • À la fin du body
  • Après l'inclusion des fichiers JS nécessaires

Utilisation avec les attributs

Une fois initialisé, vous pouvez utiliser les attributs suivants sur vos éléments HTML :

Attribut Description Exemple
data-tooltip ou tf-tooltip Contenu du tooltip <span data-tooltip="Mon tooltip">Texte</span>
data-tooltip-position ou tf-position Position du tooltip <span data-tooltip="Info" data-tooltip-position="right">Texte</span>
data-tooltip-theme ou tf-theme Thème spécifique <span data-tooltip="Info" data-tooltip-theme="dark">Texte</span>
data-tooltip-content ou tf-content ID d'un élément HTML contenant le contenu <span data-tooltip-content="#monContenu">Texte</span>

Attributs disponibles

Attribut Description Exemple
data-tooltip Texte de l'infobulle <button data-tooltip="Info">
data-tooltip-position Position de l'infobulle (top, right, bottom, left) <button data-tooltip="Info" data-tooltip-position="top">
data-tooltip-theme Thème de l'infobulle (light, dark, auto) <button data-tooltip="Info" data-tooltip-theme="dark">
data-tooltip-content ID de l'élément contenant le HTML <button data-tooltip-content="#tooltip-content">
data-tooltip-toggle Active le mode toggle (clic) <button data-tooltip="Info" data-tooltip-toggle>

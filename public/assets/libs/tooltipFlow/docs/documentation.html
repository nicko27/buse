TooltipFlow Documentation

Installation

Incluez les fichiers nécessaires dans votre HTML :

<link rel="stylesheet" href="assets/libs/tooltipFlow/src/tooltipFlow.css">
<script src="assets/libs/tooltipFlow/src/tooltipFlow.js"></script>

Configuration

Initialisez TooltipFlow avec vos options :

document.addEventListener('DOMContentLoaded', () => {
    TooltipFlow.init({
        debug: false,              // Active les logs de debug
        defaultPosition: 'top',    // Position par défaut
        margin: 8,                // Marge en pixels
        showDelay: 200,           // Délai d'affichage en ms
        hideDelay: 200,           // Délai de masquage en ms
        interactive: false         // Interaction avec le tooltip
    });
});

Options

Options Globales

Option Type Défaut Description
debug boolean false Active les logs de debug
defaultPosition string 'top' Position par défaut (top, bottom, left, right)
margin number 8 Marge en pixels
showDelay number 200 Délai d'affichage en ms
hideDelay number 200 Délai de masquage en ms
interactive boolean false Permet l'interaction avec le tooltip

Attributs HTML

Attribut Description Exemple
data-tooltip Contenu simple du tooltip data-tooltip="Mon message"
data-tooltip-id ID de l'élément contenant le HTML data-tooltip-id="monTooltip"
data-theme Thème du tooltip data-theme="dark"
data-tooltip-hide-arrow Masque la flèche data-tooltip-hide-arrow="true"

Fonctionnalités Avancées

Positionnement Intelligent

Le tooltip se positionne automatiquement en fonction de :

  • L'espace disponible à l'écran
  • La position de défilement
  • La taille du contenu
  • La position de l'élément cible

Gestion des Grandes Fenêtres

  • La flèche suit toujours le centre de l'élément cible
  • Limitation intelligente pour rester dans les bordures
  • Support du scroll horizontal et vertical

Tooltips Interactifs

TooltipFlow.init({
    interactive: true,
    hideDelay: 300
});

Modes de déclenchement

TooltipFlow propose trois modes de déclenchement pour l'affichage des tooltips :

1. Configuration globale

Vous pouvez définir le mode de déclenchement par défaut lors de l'initialisation :

TooltipFlow.init({
    triggerMode: 'click'  // 'hover' (défaut), 'click', ou 'dblclick'
});

2. Configuration par élément

Vous pouvez aussi personnaliser le mode de déclenchement pour chaque élément via l'attribut data-tooltip-trigger :

<!-- Tooltip au survol (comportement par défaut) -->
<button data-tooltip="Info bulle au survol">Survol</button>

<!-- Tooltip au clic simple -->
<button 
    data-tooltip="Info bulle au clic" 
    data-tooltip-trigger="click">
    Clic
</button>

<!-- Tooltip au double-clic -->
<button 
    data-tooltip="Info bulle au double-clic" 
    data-tooltip-trigger="dblclick">
    Double-clic
</button>

Exemples interactifs

Comportement des tooltips

  • Mode hover :
    • Apparaît au survol de la souris
    • Disparaît quand la souris quitte l'élément
  • Mode click :
    • Bascule l'affichage au clic sur l'élément
    • Se ferme au clic en dehors ou au clic sur l'élément
  • Mode double-click :
    • Bascule l'affichage au double-clic sur l'élément
    • Se ferme au clic en dehors ou au double-clic sur l'élément
Notes importantes :
  • L'attribut data-tooltip-trigger a la priorité sur la configuration globale
  • Un seul tooltip peut être affiché à la fois en mode click ou double-click
  • Le clic en dehors d'un tooltip le fermera automatiquement en mode click/double-click

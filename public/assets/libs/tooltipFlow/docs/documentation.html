Documentation TooltipFlow

Introduction

TooltipFlow est une bibliothèque JavaScript légère et flexible pour créer des info-bulles (tooltips) dynamiques et intelligentes. Elle gère automatiquement le positionnement, le contenu HTML, les thèmes, et différents modes de déclenchement.

Caractéristiques principales :

  • Positionnement automatique intelligent pour rester dans la fenêtre visible.
  • Support du contenu texte simple et HTML complexe.
  • Thèmes clair (défaut) et sombre intégrés.
  • Modes de déclenchement : survol (hover), clic (click), double-clic (dblclick).
  • Option pour rendre les tooltips interactifs (survolables).
  • Configuration des délais d'affichage et de masquage.
  • Possibilité de masquer la flèche du tooltip.
  • Initialisation facile via attributs data- ou JavaScript.

Installation

Pour utiliser TooltipFlow, incluez le fichier CSS et le fichier JavaScript dans votre page HTML :

<!-- Feuille de style de TooltipFlow -->
<link rel="stylesheet" href="path/to/tooltipFlow.css">

<!-- Script de TooltipFlow -->
<script src="path/to/tooltipFlow.js"></script>
Note : Assurez-vous que le chemin vers les fichiers tooltipFlow.css et tooltipFlow.js est correct par rapport à l'emplacement de votre fichier HTML.

Initialisation

TooltipFlow s'initialise automatiquement en détectant les éléments avec les attributs data-tooltip ou data-tooltip-id. Cependant, vous pouvez personnaliser le comportement global en appelant la méthode TooltipFlow.init(), idéalement après le chargement du DOM.

document.addEventListener('DOMContentLoaded', () => {
    TooltipFlow.init({
        // Options globales (voir section Configuration Avancée)
        debug: false,
        defaultPosition: 'top', // Position par défaut si non spécifiée par data-tooltip-position
        margin: 8,
        showDelay: 100,        // Délai avant affichage (ms)
        hideDelay: 100,        // Délai avant masquage (ms)
        interactive: false,      // Le tooltip peut-il être survolé ?
        triggerMode: 'hover'     // Mode de déclenchement par défaut ('hover', 'click', 'dblclick')
    });
});

L'appel à TooltipFlow.init() n'est nécessaire que si vous souhaitez modifier les options globales par défaut. Sinon, la bibliothèque fonctionnera avec ses paramètres par défaut dès l'inclusion du script.

Utilisation de Base

Le moyen le plus simple d'ajouter un tooltip est d'utiliser l'attribut data-tooltip sur n'importe quel élément HTML.

Texte avec tooltip 
<!-- Tooltip sur un bouton -->
<button data-tooltip="Ceci est une info-bulle simple.">Survolez-moi</button>

<!-- Tooltip sur un span -->
<span data-tooltip="Une autre info-bulle !">Texte avec tooltip</span>

Tooltip avec Contenu HTML

Pour utiliser du HTML complexe dans votre tooltip, créez un élément (par exemple une div) quelque part dans votre page (vous pouvez le cacher avec style="display: none;"), donnez-lui un id unique, et référencez cet id avec l'attribut data-tooltip-id sur l'élément déclencheur.

Titre Important

Ceci est un paragraphe avec du gras et un lien.

  • Liste
  • d'éléments
<!-- Contenu HTML du tooltip (peut être caché) -->
<div id="mon-tooltip-html" style="display: none;">
    <h4>Titre Important</h4>
    <p>Ceci est un paragraphe avec du <strong>gras</strong> et un <a href="#">lien</a>.</p>
    <ul><li>Liste</li><li>d'éléments</li></ul>
</div>

<!-- Élément déclenchant le tooltip HTML -->
<button data-tooltip-id="mon-tooltip-html">Tooltip HTML</button>
Important : Vous ne pouvez utiliser qu'un seul des attributs data-tooltip ou data-tooltip-id sur un même élément. Si les deux sont présents, data-tooltip-id aura la priorité.

Configuration Avancée

Vous pouvez affiner le comportement et l'apparence des tooltips globalement via TooltipFlow.init() ou individuellement via des attributs data-.

Attributs HTML (Configuration par Élément)

Attribut Description Valeurs Possibles Exemple
data-tooltip Définit le contenu texte simple du tooltip. Texte data-tooltip="Info ici"
data-tooltip-id ID de l'élément HTML contenant le tooltip. Prioritaire sur data-tooltip. ID d'un élément HTML data-tooltip-id="monContenu"
data-tooltip-position Force une position spécifique pour le tooltip. top, bottom, left, right data-tooltip-position="left"
data-tooltip-theme Applique un thème spécifique au tooltip. light (défaut), dark data-tooltip-theme="dark"
data-tooltip-hide-arrow Masque la flèche du tooltip. true, false (défaut) data-tooltip-hide-arrow="true"
data-tooltip-trigger Définit comment le tooltip est déclenché. hover (défaut), click, dblclick data-tooltip-trigger="click"

Positionnement

TooltipFlow choisit automatiquement la meilleure position (top, bottom, left, right) pour afficher le tooltip en fonction de l'espace disponible dans la fenêtre. Vous pouvez forcer une position spécifique avec l'attribut data-tooltip-position.

<button data-tooltip="Position forcée à gauche" data-tooltip-position="left">Gauche</button>
<button data-tooltip="Position forcée en bas" data-tooltip-position="bottom">Bas</button>

La flèche du tooltip s'ajuste automatiquement pour pointer vers le centre de l'élément déclencheur, même si le tooltip est décalé pour rester visible.

Thèmes

Deux thèmes sont intégrés : light (par défaut) et dark. Appliquez un thème via l'attribut data-tooltip-theme.

<!-- Thème Clair (implicite) -->
<button data-tooltip="Thème Clair (défaut)">Clair</button>

<!-- Thème Sombre -->
<button data-tooltip="Thème Sombre" data-tooltip-theme="dark">Sombre</button>

Modes de Déclenchement

Choisissez comment l'utilisateur interagit pour afficher le tooltip :

  • hover (défaut) : Le tooltip s'affiche au survol de la souris et disparaît lorsque le curseur quitte l'élément (et le tooltip s'il est interactif).
  • click : Le tooltip apparaît/disparaît à chaque clic sur l'élément déclencheur. Il se ferme aussi si l'utilisateur clique en dehors.
  • dblclick (double-clic) : Le tooltip apparaît/disparaît à chaque double-clic sur l'élément déclencheur. Il se ferme aussi si l'utilisateur clique en dehors.

Configurez le mode globalement via TooltipFlow.init({ triggerMode: '...' }) ou par élément avec data-tooltip-trigger="...".

<button data-tooltip="Info bulle au survol" data-tooltip-trigger="hover">Survol</button>
<button data-tooltip="Info bulle au clic" data-tooltip-trigger="click">Clic</button>
<button data-tooltip="Info bulle au double-clic" data-tooltip-trigger="dblclick">Double-clic</button>
En mode click ou dblclick, un seul tooltip peut être affiché à la fois. Cliquer en dehors du tooltip actif le fermera.

Interactivité

Par défaut, le tooltip disparaît si la souris le quitte. En activant l'option globale interactive: true, l'utilisateur peut déplacer la souris sur le tooltip sans qu'il se ferme (utile pour les tooltips contenant des liens ou des boutons).

// Activation globale
TooltipFlow.init({ interactive: true });

Délais d'Affichage et de Masquage

Contrôlez le temps en millisecondes avant qu'un tooltip n'apparaisse (showDelay) ou ne disparaisse (hideDelay) après le déclenchement de l'événement (survol/sortie de souris).

// Configuration globale des délais
TooltipFlow.init({
    showDelay: 500, // Apparaît après 500ms
    hideDelay: 100  // Disparaît après 100ms
});

Masquer la Flèche

Pour un look plus minimaliste, vous pouvez masquer la flèche du tooltip avec l'attribut data-tooltip-hide-arrow="true".

<button data-tooltip="Ce tooltip n'a pas de flèche." data-tooltip-hide-arrow="true">Sans Flèche</button>

Personnalisation CSS

Vous pouvez personnaliser l'apparence des tooltips en surchargeant les styles définis dans tooltipFlow.css. Les classes principales sont :

  • .tooltip: Le conteneur principal du tooltip.
  • .tooltip-content: Le conteneur interne du contenu.
  • .tooltip::before: La flèche du tooltip.
  • .tooltip[data-position="..."]: Pour cibler une position spécifique.
  • .tooltip[data-theme="..."]: Pour cibler un thème spécifique.
  • .tooltip.visible: Classe ajoutée lorsque le tooltip est visible.
/* Exemple : Changer la couleur de fond du thème sombre */
.tooltip[data-theme="dark"] {
    background-color: #111;
    border: 1px solid #555;
}

.tooltip[data-theme="dark"]::before {
    /* Ajuster la couleur de la flèche pour le thème sombre */
    border-top-color: #111; /* Si position='top' */
    /* etc. pour les autres positions */
}
